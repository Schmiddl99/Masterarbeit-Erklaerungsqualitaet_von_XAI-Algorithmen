# Masterarbeit-Erklärungsqualität_von_XAI-Algorithmen
GitHub Repository von Julius Schmidt als Anhang zur Masterarbeit mit dem Thema "Erklärungsqualität der Darstellungen von Explainable-AI-Algorithmen"

## Anforderung zum Ausführen des Codes
1. Installation von Anaconda Navigator ist vorausgesetzt.
2. Die Verwendung eines virtuellen Environments wird dringend empfohlen. -> Hilfestellung https://medium.com/@claudia.nikel/how-to-setup-a-jupyter-notebook-in-vs-code-w-virtual-env-kernels-install-package
   1. Hinweis: Der Artikel ist für UNIX-basierte Betriebssysteme wie MacOS oder Linux. Die Aktivierung des virtuellen Environments unter Windows erfolgt mittels ``*Env_Name*\Scripts\activate`` (funktioniert nur in der CMD!)
3. Abhängigkeiten können mit ``pip install -r requirements.txt`` installiert werden. Bei Problemen löschen Sie den Cache im virtuellen Environment und folgen Sie Schritt 2 und 3 erneut.
